# React Custom Hooks

## Utilize useState

1. useInput

- input 태그에 사용할 수 있는 커스텀 훅

* 유효성 검사도 포함됨

2. useTabs

- 탭을 전환할 때 사용할 수 있는 커스텀 훅

## Utilize useEffect

1. useTitle

- title 태그 텍스트를 변경하는 커스텀 훅

2. useClick

- 클릭 이벤트일 때 변경해주는 커스텀 훅

3. useConfirm

- 유저에게 메세지를 보여주는 커스텀 훅

4. usePreventLeave

- 브라우저 창을 끄기 전에 메세지를 보여주는 커스텀 훅
